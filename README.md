# WPM Strikes Back

A fast-paced typing web application featuring a cosmic aesthetic and dual functionality. Test your typing speed in a battle against Darth Vader or check your mechanical keyboard's health with the built-in switch tester.

**[Play Live Here](https://wpmstrikesback.netlify.app/)**

---

## 🎮 Features & Modes

### 🔀 Choose Your Path
*   **Dual Modes:** A sleek main menu that allows users to seamlessly switch between the "Switch Tester" and "Typing Speed" battle mode.
*   **Cosmic UI:** The entire application features a cohesive, immersive starry background aesthetic.

### ⌨️ Switch Tester Mode
*   **Interactive 87-Key Layout:** A fully interactive virtual Tenkeyless (TKL) keyboard display.
*   **Live Visual Feedback:** Keys illuminate green instantly upon pressing to confirm your physical switches are registering properly.
*   **Progress Tracking:** Features a built-in progress bar and counter (e.g., "1 / 87 keys tested") to ensure you don't miss any keys during testing.

### ⚔️ Typing Speed (Battle Mode)
*   **Combat Mechanics:** 
    *   Type a word perfectly to launch an attack and earn **Fight Points**.
    *   Make a typo, and the current letter flashes red while your character takes a hit!
*   **Responsive Typing Detection:** Instant, lightning-fast keystroke registration perfectly synced with visual cues (white for correct, red for incorrect).
*   **Custom Match Durations:** Choose between a **30-second** sprint or a **60-second** endurance battle against Darth Vader before the fight begins.
*   **Accurate WPM Tracking:** Calculates standard typing test metrics which is displayed clearly on the post-match Results Screen.

### 🚀 General 
*   **Web-App Optimized:** Full-screen, responsive, and native-feeling UI with disabled text-selection and zooming.

## 🚀 How to Run Locally

Since the game uses standard HTML, CSS, and JS, setting it up on your local machine is incredibly simple.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/taufiqmrahman/WPMStrikesBack.git](https://github.com/taufiqmrahman/WPMStrikesBack.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd WPMStrikesBack
    ```
3.  **Run the game:**
    Simply double-click the `index.html` file to open it in your default web browser.

## 📖 How to Play

1.  Open the game and choose your path from the main menu.
2.  **For Switch Tester:** Press every key on your physical keyboard to ensure they light up green on the screen.
3.  **For Typing Speed:** Select your preferred time limit (30s or 60s) and click **Start Fight**.
4.  Look at the target word on the screen and type it as fast as you can. Correct letters will turn white.
5.  If you type a wrong letter, it will turn red, and your character will get punched by the opponent.
6.  Finish the word to strike your opponent and rack up Fight Points.
7.  When the timer ends, review your final WPM and total score!
