# 🌌 Type Fighter

An interactive, fast-paced typing speed game with an antigravity visual twist and Minecraft-style combat mechanics. Test your typing speed, hit your Words Per Minute (WPM) goals, and battle it out in a zero-gravity arena!

**[Play the Live Demo Here](https://<your-username>.github.io/<repository-name>/)**

---

## 🎮 Gameplay Features

*   **Antigravity Environment:** Target words float dynamically across the screen, simulating a zero-gravity space.
*   **Combat Mechanics:** 
    *   Type a word perfectly to launch an attack and earn **Fight Points**.
    *   Make a typo, and the current letter flashes red while your character takes a hit!
*   **Responsive Typing Detection:** Instant, lightning-fast keystroke registration perfectly synced with visual cues (white for correct, red for incorrect).
*   **Custom Match Durations:** Choose between a **30-second** sprint or a **60-second** endurance battle before the fight begins.
*   **Accurate WPM Tracking:** Calculates standard typing test metrics (characters typed / 5) displayed clearly on the post-match Results Screen.
*   **Web-App Optimized:** Full-screen, responsive, and native-feeling UI with disabled text-selection and zooming.

## 🛠️ Tech Stack

This project is built using pure vanilla web technologies. No external frameworks or libraries are required to run it.
*   **HTML5 & Canvas API** (For rendering the game loop, antigravity physics, and blocky character graphics)
*   **CSS3** (For the app-like UI, start menu, and results screen)
*   **Vanilla JavaScript (ES6)** (For game logic, typing detection, and timing)

## 📖 How to Play

1.  Open the game and select your preferred time limit (30s or 60s).
2.  Click **Start Fight**.
3.  Look at the dark/transparent floating word on the screen.
4.  Type the word as fast as you can. Correct letters will turn white.
5.  If you type a wrong letter, it will turn red, and your character will get punched by the opponent.
6.  Finish the word to strike your opponent and rack up Fight Points.
7.  When the timer ends, review your final WPM and total score!
